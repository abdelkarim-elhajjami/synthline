# Synthline
Synthetic Data Generation for Requirements Engineering using Large Language Models.

## Usage
### Cloud (Hugging Face Spaces)
The easiest way to use Synthline. No installation required.

[**Launch Synthline on Hugging Face Spaces**](https://huggingface.co/spaces/karimelhajjami/synthline)

Simply enter your **OpenAI** or **OpenRouter** API key to start generating.

### Local (Docker)
Recommended if you want to use local models (Ollama) or run offline.

```bash
git clone https://github.com/abdelkarim-elhajjami/Synthline.git
cd synthline
./deploy.sh
```
